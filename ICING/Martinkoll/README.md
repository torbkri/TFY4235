# Readme for exam assignment

There are 3 files in this project. One for Ising model without H and one with H. Also one jupyter notebook file for problem 2. The tasks are pretty straightforward to run.

## How to run

### Problem 3
For each subtask, uncomment code and run. Remember to uncomment function call to run the function. This is the same in both ising.py and ising_with_H.py. Variables such as n_sim, runs or amount of temperature values can be changed to reduce runtime, however it should not take any longer than 20 mins, at the longest. 
The data is also in the project folder, so it's not necessary to run the commented code to see the figures.

### Problem 2
To run problem 2, press "run all" and the results should be given.
