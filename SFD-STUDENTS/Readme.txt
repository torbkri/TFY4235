Signature Verification Project Overview
======================================

This repository contains supplementary material for the TFY4235/FY8904 Computational Physics assignment on signature verification:
https://www.overleaf.com/read/gbfhgfjwwwyq#7e762d

Data
----

- signatures-dataset/
	- Contains a filtered and preprocessed binarized subset of the CEDAR signature dataset.
	- filtered_dataframe_binarized.csv provides tabular metadata for the filtered datasets.

Notebooks
---------

- 5 Kernels and convolution.ipynb
	- Belongs to exam question 5: "Kernels and Convolution".
	- Demonstrates convolution-based image processing on signature images.

- 7 Optimization based alignment.ipynb
	- Belongs to exam question 7: "Optimization based alignment".
	- Implements optimization-based rotation and alignment of signature images.

Additional notebooks (e.g., preprocessing, data loading, training, and evaluation) support the end-to-end signature verification workflow based on the CEDAR dataset.
